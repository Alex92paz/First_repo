Именованные кортежи (namedtuple)

Использование кортежей в Python для передачи данных между функциями обработчиками – это хорошая
и распространенная практика. Но есть одно неудобство у кортежей – нам необходимо помнить индексацию
элементов в кортеже и не путать их порядок. Это не всегда удобно и для ситуаций, когда в кортеже
достаточно много элементов, такой подход усложняет читабельность кода.

Поэтому были придуманы именованные кортежи namedtuple в Python. Это расширение стандартного типа
данных tuple, позволяющее выполнять доступ к элементам списка по имени, а не по индексу. Что делает
наш код более легким для чтения и более понятным.

Именуемый кортеж создается с помощью функции namedtuple из модуля collections.
