Модуль collections

В Python есть пакет с дополнительными коллекциями, которые могут очень понадобиться
начинающему разработчику и значительно расширить его арсенал. Модуль collections в
Python – это встроенный модуль, содержащий специализированные типы данных, которые
предоставляют альтернативы стандартным встроенным контейнерам. Эти специализированные
типы данных предназначены для решения конкретных задач программирования.
