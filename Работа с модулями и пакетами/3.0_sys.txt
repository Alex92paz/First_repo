Модуль sys. Обработка аргументов командной строки

Модуль sys в Python является частью стандартной библиотеки и предоставляет доступ к
некоторым переменным и функциям, взаимодействующим с интерпретатором Python. Модуль
sys часто используется для получения информации о среде, в которой работает программа,
и для взаимодействия с системой на более низком уровне.

Модуль sysобеспечивает доступ к переменным и функциям, связанным с самим Python и
операционной системой, на которой он установлен. Этот модуль также содержит много
информации о системе импорта Python.

Вот несколько ключевых возможностей модуля sys:

sys.argv - список аргументов командной строки, переданных скрипту Python. Элемент argv[0]является именем скрипта, а другие элементы – дополнительные аргументы командной строки.
sys.exit() - функция выхода из Python. Вы можете передать числовой довод, который станет статусом выхода программы. Принято, что аргумент 0означает успешное завершение, а ненулевые значения указывают на ошибку.
sys.path - список строк, определяющий путь поиска интерпретатора для модулей. Вы можете изменить этот список для добавления собственных способов поиска модулей.
sys.version - строка, содержащая информацию о используемой версии Python.
sys.platform - строчка, указывающая на имя платформы, на которой выполняется Python (например, 'linux' для Linux, 'win32' для Windows).
sys.modules - словарь, содержащий загруженные модули. Ключи – это названия модулей, а значение – это объекты модулей.
