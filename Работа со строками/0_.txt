Вот основные управляющие символы, которые необходимо знать:

1. К примеру, символ \n отвечает за перенос строки (line break).
print ( "Hello\nWorld" )

2. Другой пример – горизонтальная табуляция \t (tab):
print ( "Hello\tWorld" )

3. Пример возвращения каретки ( carriage return \r).
print ( "Hello my little\rsister" )

4. Управляющий символ \b ушиба (backspace)
print ( "Hello\bWorld" )

5. Также если нам нужно выполнить выведение обратной косой черты.
print ( "Hello\\World" )

6. Чтобы экранировать одинарные и двойные кавычки и разрешить использовать кавычки внутри строчных литералов.
print ( 'It'sa beautiful day' ) 
print ( "He said, \"Hello\"" )
